ParentFlex direction

Flex direction establishes which axis should act as the main axis. It defines the direction the flex items are placed in the flex container.

Flexbox is (aside from optional wrapping) a single-direction layout concept. Think of flex items as primarily laying out either in horizontal rows or vertical columns.

To define flex direction, use: flex-direction: row | row-reverse | column | column-reverse;.

  • row (default): left to right
  • row-reverse: right to left
  • column: same as row but top to bottom
  • column-reverse: same as row-reverse but bottom to top
