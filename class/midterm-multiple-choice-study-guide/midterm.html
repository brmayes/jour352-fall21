Midterm study guide: multiple choice and short answer

This portion of the midterm will be closed book, note and internet. Below are the topics to study up on before you take the test:

HTML

What is the difference between your <head> and <body> tags?

In your head, you have meta tags like link to css, meta data and title. In the body, it's anything you can see.

What are the main tags that go in the <head>?

  • meta: used for SEO, used for responsiveness to help resizing
  • title: on the tab of the browser. good for SEO.
  • link: to link CSS to the page

What is the <meta> viewport tag for? Why is it important?

                    <meta name="viewport" content="width=device-width, initial-scale=1">

helps with screen resizing for mobile screens. don't need to exactly know the properties on this tag, do need to know why it is used.

How do you link a CSS file in your HTML?

<link rel="stylesheet" href="[style].css" type="text/css" >

What are the main tags that go in the <body>?

  • heading (h1-h6)
  • paragraph (p)
  • div
  • span
  • br
  • ul/ol
  • li
  • image (img)
  • link (a)
  • header
  • footer

How do you add an image to your HTML?

<img src="fish.jpg" alt="pic of a fish" />

What are the three different <a> types in HTML?

  • internal link to another page in same site
  • external link to a different site
  • anchor link (#description)

What property is used on the <a> tag to open a link in a new tab? Write out a link to Google using the property to open in a new tab.

This is a link to Google. Here is how it looks typed out: <a href="google.com" target="_blank" >

What are <div> and <span> tags? How are they used?

<div> tags are used as containers/wrappers -- is a block element. used to group elements and target in css. <span> tags targets specific inline elements.

CSS

What are the types of CSS implementations?

  • external stylsheet - linking a separate css file******
  • internal stylesheet - written in your head tag
  • inline - within the element tag

What are the three major types of CSS selectors?

  • id (#)
  • class (.)
  • element/tag (target li in css)

What is the box model?

element > padding > border > margin

What are some of the most-used CSS properties?

  • color
  • background-color
  • border
  • width
  • height
  • font-size
  • font-family
  • font-style
  • display
  • margin
  • padding

How do you add a border to an image? What about margins?

                img {
                    border: 1px solid black;
                    border: [width] [style] [color];
                }

What are the three ways you can format padding (and margin) values when defining different sides of an element?

                img {
                    padding: 10px; // applies 10px padding to all sides
                    padding: 5px 10px; // top/bottom then left/right
                    padding: top right bottom left; // defines individual sides
                }

What does `margin: 0 auto;` mean with reference to the question above?

No margin on the top and bottom. Right and left are auto. Used to center content on a page. Adds same amount of space on either side of block element if width is defined.

What is a hexadecimal code? How is it used?

Used to colors. #000000

What is a media query? Why is it used? Write a media query that changes the background color of your html page on small screen sizes.

Makes page responsive. Changes the styles of page/element/class/id at a defined size. Could be min-width or max-width (could be both). 

                    @media screen and (max-width: 500px) {
                        body {
                            background-color: green;
                        }
                    }

What is flexbox? Why is it used?

Flexbox is a set of CSS properties and values that are used to move elements within a container. Useful for layout of a page.

Formatting and validating

What is the value of formatting your code?

Readability. Clarity of code. Easier for others to understand what's happening in your code. See what's nested. Find mistakes/debug.

Why is it especially important to have clean, readable CSS code?

Cascading stylesheet. Styles written at the top can be overwritten by styles at bottom. Specificity helps with this.

What is a validator? Why do you use it?

Checks your code and makes sure there are no issues. Helps with debugging.
